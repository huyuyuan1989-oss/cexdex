1. ✅ **一眼看懂市場** - 所有關鍵指標即時呈現

## 🎯 戰鬥儀表板 v2.0 核心儀表

### 1️⃣ 🔴 異常警報系統 (Alert System)
**功能：偵測需要立即關注的市場異常**

| 警報級別 | 顏色 | 觸發條件 |
|---------|------|----------|
| 🔴 危險 | 紅色 | 24H TVL 急跌 >5%、CEX 資金激增 >3% |
| 🟠 警告 | 橙色 | TVL 下跌 3-5%、穩定幣佔比 >40% |
| 🟡 注意 | 黃色 | 單一公鏈暴漲 >15%、資金流入加速 |
| 🟢 正常 | 綠色 | 無異常 |

### 2️⃣ 🌡️ 市場情緒溫度計 (Fear & Greed Index)
**功能：量化市場恐懼/貪婪程度**

| 分數區間 | 情緒標籤 | 含義 |
|---------|----------|------|
| 80-100 | 🔥 極度貪婪 | 市場過熱，考慮獲利了結 |
| 65-79 | 😊 貪婪 | 樂觀情緒，可持有 |
| 55-64 | 😐 略微貪婪 | 偏樂觀 |
| 45-54 | 😶 中性 | 等待方向 |
| 35-44 | 😟 略微恐懼 | 偏悲觀，謹慎 |
| 20-34 | 😰 恐懼 | 市場恐慌，可能是機會 |
| 0-19 | 😱 極度恐懼 | 恐慌拋售，反向指標 |

**計算因子：**
- TVL 變化 (30%)
- 穩定幣佔比 (25%) - 穩定幣高 = 恐懼
- Altcoin 佔比 (25%) - Altcoin 高 = 貪婪
- 週趨勢 (20%)

### 3️⃣ ⚡ 動量雷達 (Momentum Radar)
**功能：資金流動的速度和方向判定**

| 分數 | 方向 | 含義 |
|------|------|------|
| +50 ~ +100 | 📈 加速上漲 | 強勁上升動能 |
| +10 ~ +49 | 📈 上漲 | 正向動能 |
| -9 ~ +9 | ➡️ 持平 | 動能穩定 |
| -49 ~ -10 | 📉 下跌 | 負向動能 |
| -100 ~ -50 | 📉 加速下跌 | 強勁下跌動能 |

### 4️⃣ ⏱️ 時間緊迫性指標 (Urgency Score)
**功能：告訴飛行員「還有多少時間可以行動」**

| 分數 | 機會窗口 | 建議 |
|------|----------|------|
| 8-10 | ⚡ 立即行動 | 趨勢明確，錯過可能造成損失 |
| 6-7 | 🔔 24小時內 | 建議今日內做出決策 |
| 4-5 | 📅 2-3天內 | 可觀察但需密切關注 |
| 0-3 | 🕐 從容規劃 | 市場穩定，不急於行動 |

### 5️⃣ 🎯 具體行動建議系統
**功能：不只說「買入」，告訴你買什麼、配置多少**

**輸出項目：**
- **主要行動建議** - 例如「積極進場：市場資金大量流入，把握機會建立多頭倉位」
- **倉位建議** - 例如「建議加倉至 70-100%」
- **關注公鏈** - 列出資金流入最強的前3個公鏈
- **資產配置** - 根據情緒推薦資產類型（Altcoin/原生幣/穩定幣）

## 🏦 CEX + DEX 整合分析

### 數據整合
| 指標 | 說明 |
|------|------|
| 全市場總資金 | CEX + DEX 的 TVL 總和 |
| CEX 佔比 | 交易所持有資金百分比 |
| DEX 佔比 | 鏈上 DeFi 資金百分比 |
| 資金流向判定 | 資金是流入/流出 CEX 還是 DEX |

### 流向解讀
| 模式 | 解讀 |
|------|------|
| CEX↑ + DEX↓ | 資金回流交易所，可能準備賣出 |
| CEX↓ + DEX↑ | 提幣參與 DeFi，市場活躍 |
| CEX↓ + DEX↓ | 資金流出市場，整體萎縮 |
| CEX↑ + DEX↑ | 新資金入場，整體增長 |
